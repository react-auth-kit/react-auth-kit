# Installation

## Using Package Manager

### Using NPM
```bash
npm install --save react-auth-kit
```

### Using Yarn
```bash
yarn add react-auth-kit
```

## Using CDN

### Unpkg
```bash
https://unpkg.com/react-auth-kit/dist/index.umd.js
```

### JsDelivr
```bash
https://cdn.jsdelivr.net/npm/react-auth-kit/dist/index.umd.js
```
